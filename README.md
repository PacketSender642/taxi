# Taxi

simple compiled language written in V

## Building

You must have a V compiler. (<vlang.io>)


```bash
v .         # for a simple build
v . -prod   # for a production build
```

## Samples 

```taxi

import passanger.io

fun main() {
  io.println("Hello, Taxi.")
}

```
