# Taxi

a event driven compiled language written in V

## Building

You need a V compiler (vlang.io)
```bash
make -j2
```

## Samples 

```taxi
on run { # kinda like the main function!
    puts("Hello taxi!");
}
```

## Platforms supported currently

- [ ] Linux
- [ ] NetBSD

We wont add windows support.
